## Symbol 是什么？

- Symbol 是 ES6 引入的一种新的原始数据类型，
  - 它的特点是每一个 Symbol 值都是唯一的，
  - 通常用于作为对象属性的 key，从而避免属性命名冲突。

- Symbol 属性不会被 for...in、Object.keys 等方法遍历，
  - 如果需要获取 Symbol 属性可以使用 Object.getOwnPropertySymbols。

此外 Symbol.for 可以创建全局共享 Symbol

## position 有哪些值？

position 有五种属性值：

- static：默认定位，元素按照正常文档流排列  
- relative：相对定位，相对于自身原位置偏移, 可以使用 top, bottom, left, right 等属性进行偏移
- absolute：绝对定位，相对于最近的定位祖先元素  
- fixed：固定定位，相对于浏览器窗口定位  
- sticky：粘性定位，在滚动时会从 relative 变为 fixed

## 如何判断数组？

### 判断数组常见有四种方式：

1. 使用 `Array.isArray(arr)` —— 最推荐  
2. 使用 `arr instanceof Array`
3. 使用 `Object.prototype.toString.call(arr) === "[object Array]"`
4. 使用 `arr.constructor === Array`（不推荐）

### typeof 和 instanceof 的区别

|区别|typeof|instanceof|
|:--:|:--:|:--:|
|判断类型|基本数据类型|对象类型|
|返回值|string|boolean|
|适用场景|number、string 等|Array、Date、RegExp 等|

### 总结

`typeof` 适合判断基本类型
`instanceof` 用于判断对象类型
`Array.isArray(arr)` 是判断数组的推荐方法
`Object.prototype.toString.call(arr)` 是最准确的类型判断方式

## JavaScript 隐式类型转换

- 当运算符左右两侧操作数类型不同时，会触发隐式类型转换
- 转换规则如下：
  - 调用 valueOf 方法，返回原始值
  - 调用 toString 方法，返回字符串
  - 调用 Symbol.toPrimitive 方法，返回原始值,优先级最高

### 题目代码

```javascript
var obj = {
  value: 0,
  valueOf() {
    return 1;
  },
  toString() {
    return '2'
  },
  [Symbol.toPrimitive]() {
    return 3
  }
}

console.log(obj + 1); // 4
```

执行顺序

- 当对象参与运算时，例如：`obj + 1`

- JavaScript 会执行 对象转原始值（ToPrimitive）。

转换顺序是：

1. Symbol.toPrimitive
2. valueOf
3. toString

但注意：

- 只要某一步成功返回原始值
- 后面的步骤就不会执行

## new构造调用的过程

无论是通过字面量还是通过new进行构造函数调用创建出来的对象，其实都一样。调用new的过程如下：

1. 创建一个新对象
2. 原型绑定
3. 绑定this到这个新对象上
4. 返回新对象
